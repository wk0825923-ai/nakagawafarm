畝（ロット）単位で農作業・農薬・施肥・収穫・GAP書類を管理する農家向けWebアプリ

レタス・とうもろこし・米／技能実習生含む小規模法人向け

本番: syatyo-suport.vercel.app React 18（ビルドなし）+ Supabase 農場データは localStorage 保存 全 push 済み

A 毎日〜週次で使う「記録・確認」系（メイン導線）

左メニュー上段。使用頻度順（毎日→季節ごと→稀）に配置。

毎日

総合ダッシュボード

月次サマリー・今日のタスク・直近記録・GAP進捗・収穫リスク警告・作物提案・圃場カード一覧。

毎日

日報入力

4ステップ入力（日付/圃場→作業内容+写真→農薬/施肥→確認・保存）。複数圃場の一括記録対応。写真圧縮・音声転記・二重登録ガード付き。

毎日

日報管理

記録の一覧／閲覧／編集／削除。

週次

圃場まとめ

ロット別の生産履歴を表形式で表示。CSV出力（数式インジェクション対策済み）。

週次

収穫予測

積算温度ベースで収穫時期を予測。

随時

出荷記録

収穫→ストック→出荷の流れとストック残管理。出荷先マスタ付き。

突合せ

整合性チェック

入力ミス／食い違いを点検。該当記録へ一発ジャンプ、収穫↔出荷の増減を要確認扱い。

B 圃場詳細（圃場ごとのタブ画面）

各圃場を開くと出るタブ群。作業の実データはここに溜まる。

圃場ダッシュボード / 畝マップ

圃場サマリーと Leaflet 畝マップ（緯度経度未設定でもクラッシュしない）。ロット追加・編集、ロット別リスク警告。

農薬散布・施肥・収穫

ロット単位で記録（各 Form / List / Section）。二重登録ガード対応。

作付サイクル・評価

作付サイクル選択と履歴、米の生育ステージタイムライン、圃場評価タブ。

C 季節ごと・監査時

季節

作付計画 / 経営予測

作付スケジュールと経営見込みの試算。

監査時

GAP帳票出力

GAP書類の PDF / Excel 出力（HTMLエスケープ済み＝XSS対策）。

監査時

GAPチェックリスト

GAP項目の点検。

D マスタ・管理（下段「管理・設定」・既定で折りたたみ）

初期設定中心で頻度は低い。関連機能はタブ統合済み。

マスタ管理

農薬／肥料／作物カテゴリをタブ統合。在庫ウィジェット・仕入れ／使用履歴付き。

スタッフ管理

スタッフ一覧＋ビザ管理。

技能実習生 作業日誌

実習生ごとの作業日誌。

多言語マニュアル

多言語対応のマニュアルライブラリ。

機器予約 / 機械整備記録

機器予約と機械整備記録（GAP機械管理・直アクセス）。

収益シミュレーター

収益の試算ツール。

設定

各種システム設定。

E 基盤・横断機能

認証・テナント

Supabase で認証と組織／農場情報を管理。農場データ本体は localStorage 保存（本番化の最重要論点）。

エラー堅牢化

統一トースト通知＋ ErrorBoundary ＋未捕捉エラーの受け皿。読込・保存失敗を握り潰さず可視化。

UX演出・共通部品

保存演出（紙吹雪）、共通削除確認モーダル、不足項目のトースト案内。

データ整合・修正済み

日付のUTC/JSTズレ修正、圃場番号の法人間分離漏れ修正。

QAハーネス

実ブラウザ E2E テスト同梱（qa/・puppeteer-core）。全ページ巡回・20圃場シナリオ。
